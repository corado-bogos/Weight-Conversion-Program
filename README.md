# weight_conversion_program
This is a beginner-friendly Python project that converts weight between kilograms (kg) and pounds (lbs).

The program:
• Asks the user to input their weight
• Asks the user to choose the unit (kilograms or pounds)
• Converts the value based on the selected unit
• Displays the converted result rounded to 2 decimal places
• Handles invalid input for the unit

This project helps practice:
• User input handling
• Conditional statements (if/elif/else)
• Basic arithmetic operations
• Simple program structure in Python
